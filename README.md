# PythonProject02
Телеграм бот на питоне3 с  помощью aiogram2

Телеграм agile ежедневник с многими разделами: заметки, привычки, ритуалы, вишлист, идеи и спринт
Логика ежедневника взял у автора(по книжке про успешный успех)
В каждом разделе есть кнопочки по типу удалить_запись, сделать запись, редактировать запись, просмотреть_все_записи все это подключено к базе данных.
Рекомендую запустить у себя и потыкаться, там будет все понятно

Вообще тут много, что надо доделывать. Возможно сокращю функционал, чтобы сделать меньше, но качественнее. После залива и до ревью буду доделывать проект
